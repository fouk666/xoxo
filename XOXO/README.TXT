0.Для запуска сервера необходим интерпретатор python, который можно скачать по ссылке:
https://www.python.org/ftp/python/3.6.0/python-3.6.0.exe
0_1. После установки интерпретатора необходимо загрузить библиотеки flask и flask-socketio.
Запустим консоль и выполним следующие команды:
\путь до папки\Python\Python36-32\python.exe \путь до папки\Python\Python36-32\Scripts\pip3.6.exe install flask
\путь до папки\Python\Python36-32\python.exe \путь до папки\Python\Python36-32\Scripts\pip3.6.exe install flask-socketio

1. Запускаем локальный сервер
python.exe run.py

2. Для того, чтобы попасть на страницу с игрой, необходимо в браузере перейти по адресу
http://127.0.0.1:5000

На странице с игрой находятся:
 - название игры в верхней части страницы
 - игровое поле по центру
 - слева расположились правила игры
 - в правой стороне находится игровой чат с системными оповещениями

Приятного пользования!
